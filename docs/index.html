What it does

PII in. Tokens out. Originals back.

The LLM receives readable context — just not the actual sensitive values.

Features

Built for production.

Everything you need to protect user data in LLM pipelines.

🔍

Hybrid detection

Regex for structured types (cards, IBANs, emails). Local Ollama for contextual types (names, addresses, national IDs). Best of both.

🏠

100% local

Detection runs on your own infra via Ollama. Raw data never leaves your network — not even for the detection step.

🔐

Encrypted vault

Token↔value mappings stored in AES-256-GCM encrypted SQLite. Keys never leave the process.

🆔

Your own IDs

Use context_id from your domain (contact_123, zoom_call_789) instead of tracking foreign UUIDs.

⚡

Parallel processing

Large texts chunked and processed in parallel goroutines. Single-pass Aho-Corasick detokenization.

📦

Dual-use

Run as a standalone HTTP service or import as a Go library. One Docker Compose command to start everything.

# 1. Tokenize before sending to LLM
resp = requests.post("http://localhost:8080/v1/tokenize", json={
    "text": transcript,
    "context_id": f"zoom_{call_id}"
})
tokenized = resp.json()

# 2. Send tokenized text to your LLM — real PII never transmitted
llm_response = openai.chat.completions.create(
    messages=[
        {"role": "system", "content": "Summarize this call transcript."},
        {"role": "user",   "content": tokenized["tokenized_text"]}
    ]
)

# 3. Detokenize the response
final = requests.post("http://localhost:8080/v1/detokenize", json={
    "text": llm_response.choices[0].message.content,
    "context_id": f"zoom_{call_id}"
})
print(final.json()["original_text"])
import "github.com/yemrealtanay/llm-hasher/pkg/hasher"

h, err := hasher.New(
    hasher.WithOllama("http://localhost:11434", "llama3.2:3b"),
    hasher.WithVault("data/vault.db", ""),
)
defer h.Close()

// Tokenize
result, err := h.Tokenize(ctx, transcript, "zoom_call_789", nil)
// result.Text → tokenized transcript, safe to send to any LLM

// Detokenize after LLM response
original, err := h.Detokenize(ctx, llmResponse, "zoom_call_789")

Get started

Up in 60 seconds.

Docker Compose handles everything — Ollama, model download, service startup.

01 — Clone

Get the repo

git clone github.com/yemrealtanay/llm-hasher
02 — Start

One command

make docker-up

Pulls Ollama + llama3.2:3b (~2GB), starts on port 8080.

03 — Verify

Health check

curl localhost:8080/healthz

Returns {"status":"ok"}

Stop sending PII to LLMs.
